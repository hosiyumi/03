使用方法：
下载Assets文件夹，复制并粘贴到自己的测试工程的Assets文件夹即可

README文件位于Assets内

以下是总体管理框架：
该工程于“资产”下分为“美术”文件夹，“脚本”文件夹和“音乐”文件夹

以下为代码和脚本管理框架：
代码和脚本位于“资产”下的“脚本”文件夹内，一个系统由多个系统组成，每个系统遵循EC架构（实体，组件分离）
不继承monobehavior的脚本皆为实体，继承monobehavior的脚本皆为组件
例：
组件A\实体
系统A\组件
系统A\子系统A实体
系统A\组件
